## ** PHONE BOOK APP **

Phone book is ReactJS app that works like a real phone book. You can store, view or search contacts in your phone book.


### **TECHNOLOGIES**

HTML5, CSS3, JavaScript (ES6), React, Bootstrap for React, Npm
